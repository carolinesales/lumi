# Lumi 

**O que é o app e o que ele se propõe a fazer:**

O Lumi é um aplicativo web mobile-first desenvolvido como Trabalho de Conclusão de Curso no Bacharelado em Sistemas de Informação da PUCPR, orientado pela Prof.ª Kelly Rafaela Otemaier. O app se propõe a simplificar e personalizar a rotina de cuidados capilares, o usuário responde um questionário, recebe um diagnóstico com Hair Score, recomendações de tratamentos e um cronograma semanal gerado automaticamente.

**Status do projeto:**

🚧 Em desenvolvimento — versão 1.0

**O que está implementado até agora:**

- Cadastro e autenticação com verificação de e-mail
- Questionário de diagnóstico capilar em 4 etapas
- Motor de recomendação por eixos ponderados
- Cálculo do Hair Score e geração de diagnóstico
- Cronograma semanal personalizado
- Histórico de cuidados com gráfico de evolução
- Dashboard com acompanhamento contínuo

**Stack:**

React · Vite · Firebase · Recharts · Font Awesome
